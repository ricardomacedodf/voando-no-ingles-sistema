# Voando no Inglês

Aplicação web para estudo de inglês com:
- login com Google
- sincronização por usuário
- armazenamento em nuvem com Supabase
- flashcards
- quiz
- combinações
- gerenciador de palavras e frases
- importação JSON
- acompanhamento de progresso

## Rodar localmente

### Pré-requisitos
- Node.js instalado
- npm instalado
- projeto Supabase configurado
- arquivo `.env` preenchido

## Instalação

1. abra a pasta do projeto
2. instale as dependências:

```bash
npm install
